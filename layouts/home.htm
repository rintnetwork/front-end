Messenger

Rint Network is categorized as a mobile and desktop operating system and decentralized browser that incorporates a messaging system. As such, Rint Network allows you to interact with a network at any time, from anywhere. It was originally released in April in 2021. It is a light client Tron Network node, and has the ability to give you access to all Tron Network decentralized applications (otherwise known as DApps) from an app that is installed on your mobile phone or tablet. This means that users can send encrypted messages as well as access decentralized applications, including a cryptocurrency wallet.

Exchange

Launching in June 2021, RintExchange is a DeFi aggregator and a decentralized exchange with smart routing. The core protocol connects a large number of decentralized and centralized platforms in order to minimize price slippage and find the optimal trade for the users. RintExchange provides a variety of features in addition to swaps. Users can trade via limit orders, deposit funds into lending protocols, move coins between different liquidity pools, and this list expands constantly.

FAQ

What is Rint Network?

Rint Network is free and open-source project for enabling anonymous, encryption & communication by directing Internet traffic through a worldwide, volunteer overlay network consisting of more than ten thousand relays in order to conceal a user's location and usage from anyone conducting network surveillance or traffic analysis.

What is the 80:10:10 Rewards Sharing Model?

The 80:10:10 reward model is a representation of the community-first ideology that Rint Network embraces. Participation with Rint Network entitles an individual to their proportional share of 80% of RNT distributed daily. 10% is allocated for the Rint Developer Fund, and 10% is allocated for Rint Networkâ€™s operating expenses.

What is RNT token?

RNT is a representation of the rewards created through community participation across all blockchains on which Rint Network operates. RNT can be exchanged at any time for its pegged value. RNT provides special benefits to holders, such as rewards from funded RintGerminator projects. RNT automatically participates on each blockchain on which Rint Network is represented. Holding RNT earns more RNT every day.

Learn more about RNT here.

How do I earn RNT Token?

RNT is earned by community participation such as voting or staking on Tron Foundation blockchain on which Rint Network operates. Currently the maximum number of RNT issued to participating community members is 1 RNT for each 1,000 TRX daily. RNT automatically participates on Tron Foundation blockchain on which Rint Network is represented. Holding RNT will earn more RNT every day.

I have some RNT. What do I do now?

No further action is required! RNT automatically participates on Tron Foundation blockchain on which Rint Network operates. Simply by holding RNT, it will automatically start accruing more rewards each day from represented Tron Foundation blockchain. In addition, boost RNT liqudity on JustSwap give apportunity to earn fees from each RNT exchange. RNT can be exchanged for its pegged value at any time, which is constantly increasing. What is RNT pegged value?

Rint Network earns node rewards from Tron Foundation blockchain on which it operates and distributes them back to the community as RNT using the 80-10-10 model. RNT is pegged to these node rewards. This means each RNT token is always redeemable for a specific amount of coins or tokens. At a minimum, 10% of all node rewards earned by Rint Network each day will be utilized to increase the pegged value of all circulating RNT. This means every RNT held goes up in redemption value every day. RNT is redeemable for its pegged value at any time. For more information and to see the current pegged value, click here.

What is Participation Power?

Participation Power is the added advantage holding RNT has over participating with a native coin. The reason for this is that Rint Network accounts can participate o represented blockchain, but are ineligible to receive the rewards for this participation. Instead, those rewards are distributed among RNT holders in proportion to how much they hold. Participating with RNT earns more rewards than participating with the same amount of coins backing each RNT.

Where can I buy RNT tokens?

Rint Network does not sell any RNT tokens. They can only be earned directly from Rint Network by community participation such as voting or staking. However, trading pairs are available on various exchanges. Some options are JustSwap and Poloniex on the Tron Foundation blockchain.

What is RintGerminator?

RintGerminator is a community-powered initiative to fund blockchain developers. This platform is a stage for developers to present DApps, tokens and technology, and build their own communities around an associated project. Community members who find sufficient value in a project can choose to directly support it by funding it with their RNT tokens and receive special community incentives. Please visit the RintGerminator website for more details.
